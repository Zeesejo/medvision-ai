# UI — Clinical Web Interface

A lightweight web interface to demo model predictions, explanations, and fairness metrics.

## Tech Options

- **Gradio** — Fast prototyping, easy to deploy
- **Streamlit** — Dashboard-style UI
- **React + FastAPI** — Production-grade clinical interface

## Planned Features

- Upload or capture medical image
- Display model prediction with confidence score
- Show explanation overlays (Grad-CAM, attention, prototypes)
- Fairness metric dashboard (subgroup performance)
- Uncertainty estimation display
